CSS 1

O Cascading Style Sheets, ou Folha de Estilo em Cascatas (CSS) foi proposto pela primeira vez em Outubro de 1994, por Hakon Lie. Ele foi ganhar destaque por meados de 1998, porém sua criação veio um pouco antes, em 1995, um grupo de empresas do ramo da informática, W3C, desenvolveu o CSS1. Antes de sua criação, o HTML era a única base para a criação de sites, na época de popularização do CSS1, o HTML estava na sua versão 4.0, que até então, era a única maneira dos programadores adaptarem ou personalizarem o layout de sua página. Sua principal função é facilitar a programação da aparência das web páginas, transformando o HTML em uma linguagem mais para a estruturação.

Seleção de Elementos

Seleção de elementos no CSS permite identificar e estilizar partes específicas do HTML. Isso pode ser feito através de diferentes seletores, como os seletores de ID, que são usados para estilizar um único elemento específico; seletores de classe, que permitem estilizar múltiplos elementos com a mesma classe; e seletores descendentes, que aplicam estilos apenas a elementos dentro de um determinado pai.

Modelo de Caixa (Box Model)

O Box Model define como os elementos ocupam espaço e interagem com outros na página. Ele é composto por diferentes camadas: o padding, que cria um espaço entre o conteúdo e a borda do elemento; a border, que é a linha ao redor do elemento e pode ter diferentes espessuras e estilos; e a margin, que define o espaço externo, separando o elemento dos demais ao redor.

Tipografia e Cores no CSS

A tipografia e as cores no CSS permitem a personalização do texto. É possível definir fontes específicas usando font-family, ajustar o tamanho e o peso das letras com font-size e font-weight, além de controlar o espaçamento entre caracteres e linhas com letter-spacing e line-height.

Essa foi a atualização mais revolucionária do CSS, lançada entre o final dos anos 2000. Ela trouxe uma série de mudanças significativas, especialmente com foco na responsividade (forma como a página se molda a diferentes tamanhos), permitindo que os sites se adaptassem de forma mais eficiente a diferentes tamanhos de tela e dispositivos. Essa evolução marcou um grande avanço no desenvolvimento web, tornando as interfaces mais flexíveis e dinâmicas.

Flexbox

O Flexbox, um novo meio de estruturar um contâiner, fazendo uma forma melhor e mais dinâmica de manipular seus elementos dentro do container, porém é unidimensional.

Grid Layout

semelhante ao grid, mas bidimensional, ou seja, ele manipula tanto linhas, quanto colunas, sendo um Flexbox, para estruturações mais “complexas”. Para estruturar você define o tamanho e quantidade, tanto de colunas quanto de linhas, depois define item por item a quantia que ele ocupará.

Media Querries

Media Queries, a principal ferramenta para a responsividade, ela permite que coloque aplicações especificas para específicos tamanhos de tela.
